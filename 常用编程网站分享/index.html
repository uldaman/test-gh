分享下我日常编程过程中常访问的几个网站

其实最常用的永远是 Google

工具类（转化生产力）

  • Stack Overflow
  • GitHub
  • Gist, GitHub 的子站，与 GitHub 不同，Gist 被用来分享一些小的代码片段
  • Hacker News，虽然是个新闻社区，实际上也可以用它搜索到很棒的解决方案
  • C-Sharp Corner，综合社区，有行业动态，也有问答
  • Code Project，开源代码分享，在上面可以找到很多好用的 Library
  • Pudn，国内版 Code Project，比较乱，但也能找到好用的资源，一些需要积分才能下载

社区门户（了解行业）

闲着没事时会去逛一逛，了解行业最新动态，follow 大佬

  • Dev
  • DZone
  • Reddit Programming
  • Diff，综合博客，互联网各地大佬的博客合集
  • InfoQ
  • 掘金
  • 开发者头条
